# Train and evaluate clustering models

This folder holds learning materials and Jupyter notebooks for clustering exercises.

- Microsoft Learn module: https://learn.microsoft.com/en-us/training/modules/train-evaluate-cluster-models/

Notebooks (exercises):

- Exercise_Train_and_evaluate_a_clustering_model.ipynb — Apply clustering algorithms and evaluate results
- Exercise_Train_and_evaluate_advanced_clustering_models.ipynb — Explore DBSCAN, spectral clustering, and others

Notes
- Use Git Bash to activate the `.venv` and run `jupyter lab`.
